# Specs — index

This folder points to specification documents that describe API, DB, frontend contracts and UI/UX.

Files:

- `design-plan/specs/api-spec.md` — API routes and contracts.
- `design-plan/specs/db-spec.md` — Database schema and migration details.
- `design-plan/specs/frontend-spec.md` — UI contracts and component expectations.
- `design-plan/specs/ui-ux-spec.md` — UX guidelines and wireframes.

Use these when implementing or testing API / DB integrations.
