## Onboarding Timeline

1. **Day 1**
   - Access repos, set up environment
   - Run tests locally
   - Deploy locally & hit health endpoint

2. **Days 2-3**
   - Shadow code reviews
   - Fix a documentation typo or small bug
   - Explore API via Postman

3. **Days 4-5**
   - Pick a small ticket (docs/tests)
   - Pair program on a feature
   - Present learnings in stand-up
