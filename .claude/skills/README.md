# MyPortFolio Claude Skills

This directory contains project-specific Claude Code skills for frontend work.

## Scope Policy

- Focus only on Track 1 and Track 2 for this cycle.
- Do not implement Track 3 features.
- Keep changes small and file-scoped.
- Respect mobile responsiveness and accessibility requirements.

## Available Skills

1. `frontend-identity-revamp`
- Use for implementing Home hero and global theme updates aligned with persona and architecture.

2. `frontend-qa-regression`
- Use for validating build/typecheck, responsive behavior, and accessibility regressions.

## Conventions

- All skill documents are English-only.
- Every skill must have valid YAML frontmatter with `name` and `description`.
- Keep `SKILL.md` concise and put detail in `references/`.
